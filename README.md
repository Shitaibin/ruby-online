## Ruby Online

This is the source code of [Ruby Online](http://) website.

## Requirements

* Ruby 2.2.0 +
* Rails 4.2.0 +
* SQLite 3.8.2 +

## Install

```bash
git clone git@github.com:Shitaibin/ruby-online.git
cd ruby-online
.bin/setup
rails s
```
